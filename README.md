This project is an interactive Movies Sales Analysis Dashboard built using Microsoft Power BI to analyze movie sales performance across different categories, regions, and time periods.
The dashboard provides actionable insights into revenue trends, top-performing movies, customer purchasing behavior, and sales distribution, enabling data-driven decision-making.

Tools & Technologies
Power BI Desktop (.pbix)
Power Query (ETL & Data Cleaning)
DAX (Data Analysis Expressions)
Data Modeling (Relationships, Star Schema)

Dashboard Components
KPI Cards (Total Revenue, Total Orders, Profit, Quantity Sold)
Bar & Column Charts for category comparison
Line Charts for time-series analysis
Pie/Donut Charts for distribution insights
Slicers for year, region, and category filtering

Future Enhancements
Integration with real-time sales data
Advanced forecasting using DAX
Deployment to Power BI Service for cloud sharing
